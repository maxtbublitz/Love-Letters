Dear Bash,

I love when we explore together. We can cd into any directory we want, and once we're there, you show me the way with your ls command, which lists all of the directories contents. We can even make our own directories together ;) using mkdir. If I'm ever lost you help me find myself with the pwd command.

Your touch is what really makes the magic happen. If I touch a file that doesn't yet exist you will create it for me. From there I can nano into that file and edit it from your elegantly simple interface. There's nothing we can't do together.

You always have the answer. If I don't understand how to use a command I can type man and you will pull up a manual on how to use it. You really take the strain off of my fingers by auto-completing what I'm typing when I use the tab key and the control + a and control + e shortcuts which take me to the start or end of a line.

Your ditto command lets me copy a file into a different directory. Normally I hate playing games but they're fun with you, all I have to do is type Fn+F10 followed by t and g.
